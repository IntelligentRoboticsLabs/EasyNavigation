Detailed Description

ROS 2 lifecycle node that manages localization for the Easy Navigation system.

This node provides the interface between the localization module in EasyNav and the ROS 2 ecosystem. It handles lifecycle transitions, real-time scheduling of periodic tasks, and parameter setup.
