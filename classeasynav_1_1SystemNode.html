Detailed Description

ROS 2 lifecycle node that coordinates all major modules of the Easy Navigation system.

This node manages the lifecycle, scheduling, and interactions between the Controller, Localizer, Planner, Maps Manager, and Sensors modules, providing a unified interface to ROS 2.
